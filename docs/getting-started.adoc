= Getting Started
:toc:
:sectanchors:
:source-highlighter: rouge

== Base URL

----
https://api.fragmea.com
----

== Authentication

All endpoints except `GET /health` require a bearer token:

[source,bash]
----
export FRAGMEA_API_KEY='...'
----

== Health Check

[source,bash]
----
curl https://api.fragmea.com/health
----

Expected response:

[source,json]
----
{"status":"ok"}
----

== First Authenticated Request

[source,bash]
----
curl https://api.fragmea.com/corpus/metadata \
  -H "Authorization: Bearer $FRAGMEA_API_KEY"
----

== Basic Legal Search

[source,bash]
----
curl https://api.fragmea.com/search \
  -H "Authorization: Bearer $FRAGMEA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "náhrada škody",
    "limit": 100,
    "topRowsLimit": 8
  }'
----

The response creates a search workspace with:

- session metadata
- a decisions dataframe reference
- a hits dataframe reference
- inline `top_rows` preview data

`candidate_count` means how many final ranked decision rows are available before inline preview truncation by `topRowsLimit`.

== Load a Stored Session

[source,bash]
----
curl "https://api.fragmea.com/session/<sessionId>" \
  -H "Authorization: Bearer $FRAGMEA_API_KEY"
----

== Decision Detail

Request only the fields you need through repeated `select` query parameters.

[source,bash]
----
curl -G "https://api.fragmea.com/decision/<decision_id>" \
  -H "Authorization: Bearer $FRAGMEA_API_KEY" \
  --data-urlencode 'select=decision_id' \
  --data-urlencode 'select=source.full_text' \
  --data-urlencode 'select=source.pdf_url'
----

== Decision Fulltext Search

[source,bash]
----
curl https://api.fragmea.com/decision/<identifier>/fulltext-search \
  -H "Authorization: Bearer $FRAGMEA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "přípustnost dovolání",
    "mode": "phrase",
    "limit": 10,
    "contextChars": 120
  }'
----

== Decision Semantic Search

[source,bash]

  -H "Content-Type: application/json" \
  -d '{

== Search Within Existing Decision Slice

[source,bash]
----
curl https://api.fragmea.com/decisions/search-within-dataframe \
  -H "Authorization: Bearer $FRAGMEA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "ref": "nsoud-search-session:123:decisions",
    "query": "promlčení nároku",
    "limit": 50,
    "topRowsLimit": 6
  }'
----

== Typed Decision Aggregate Example

[source,bash]
----
curl https://api.fragmea.com/decisions/aggregate \
  -H "Authorization: Bearer $FRAGMEA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "ref": "nsoud-search-session:123:decisions",
    "groupBy": [{"column":"decision_date","datePart":"year","label":"year"}],
    "metrics": [{"operation":"count","alias":"decision_count"}],
    "limit": 20
  }'
----

For exact field names, enums, and request body shapes, use `openapi/fragmea-api.openapi.yaml` as the source of truth.
